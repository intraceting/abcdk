## 授权结构

|Version |Category |Product |Begin   |Duration  |Node    |Reserve |
|:-------|:--------|:-------|:-------|:---------|:-------|:-------|
|1 byte  |1 byte   |1 byte  |8 Bytes |2 Bytes   |2 Bytes |2 Bytes |

* Version： 版本号。
* Category： 类别
* Product： 型号。
* Begin：启用时间(秒，UTC)。
* Duration：有效期限(天)。
* Node：节点数量(个)。
* Reserve：预留。
