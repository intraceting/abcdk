
## ABCDK

* A better c development kit. 

### License

* ABCDK is MIT-licensed.

### 查看编译帮助

```bash
configure.sh -?
```
### 编译和安装

```bash
configure.sh [ ... ]
make
sudo make install
```

### 编译和打包

```bash
configure.sh [ ... ]
make
sudo make package
```

### 目录说明

- abcdkcomm 网络通讯。
- abcdkutil 基础工具。
- tests 测试样例。
- tools 自定义工具。
